POS Disable Order Delete Access (User-Based)
This module adds an extra layer of control to the Point of Sale by restricting the ability to delete POS orders. A new boolean option is added in the User configuration. When enabled for a user, the delete option will be hidden in the POS Orders screen, preventing the user from deleting any POS order.

Detailed Features List

User Configuration for Delete Access Control

  • Users can be configured to enable or disable their permission to delete POS orders.
  • A dedicated boolean field in the user settings allows administrators to decide who is allowed to delete orders.

Automatic Removal of Delete Option in POS Orders

  • When a user is restricted, the delete icon in the POS Order interface automatically becomes invisible.
  • This ensures a clean user experience while enforcing strict delete permissions at both UI and backend levels.
